# HiddenJailbreak

A Bypass Jailbreak Detection Tweak!

## How to build
1. Install [Theos](https://github.com/theos/theos)
2. git clone `https://github.com/noobpk/HiddenJailbreak`
3. `cd hiddenjailbreak`
4. `make package`
